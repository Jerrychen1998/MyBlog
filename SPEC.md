# 个人博客网站 - 详细规范文档

## 1. 项目概述
- 项目名称：个人主页 + 在线交互式简历 + 多标签云端文本编辑器
- 技术栈：Vue3 + Vite + Element Plus + TailwindCSS + SortableJS + GitHub JSON云端存储
- 部署目标：GitHub Pages
- 双端适配：PC端（优先）+ 移动端自适应

## 2. 全局规范

### 2.1 主题系统
- 浅色模式：主色 #f5f5f5（浅灰），深蓝 #1e3a5f，辅助色 #3b82f6（浅蓝）
- 深色模式：主色 #1a1a2e，深蓝 #3b82f6，辅助色 #60a5fa
- 自动跟随系统主题，支持手动切换
- localStorage 存储主题偏好

### 2.2 导航栏
- PC端：固定顶部，高度60px，显示完整导航文字
- 移动端：折叠为汉堡菜单，高度50px
- 包含GitHub仓库链接图标

### 2.3 响应式断点
- PC端：≥1200px
- 平板：768px - 1199px
- 手机端：≤767px

## 3. 页面详情

### 3.1 首页
- 头像：圆形，PC端120px，移动端80px
- 个人名称：PC端28px，移动端22px
- 标签：圆角标签样式，PC端14px，移动端12px
- 底部卡片：PC端横向排列，移动端纵向排列

### 3.2 个人简历页
- 模块：基础信息、职业经历、项目经验、技能图谱、附加信息
- 卡片式布局，模块可展开/折叠
- 支持复制、导出PDF

### 3.3 多标签云端文本编辑器
- 标签栏：支持拖拽排序（PC端）、折叠菜单（移动端）
- 编辑区：自适应高度，支持格式编辑
- 云端同步：GitHub JSON文件存储

## 4. 云端数据结构
```json
{
  "id": "UUID",
  "title": "文本标题",
  "content": "文本内容",
  "createTime": "时间戳",
  "updateTime": "时间戳",
  "isArchive": true
}
```

## 5. 验收标准
- npm run dev 可正常启动
- 所有页面功能正常运行
- 双端适配无错乱
- 云端同步逻辑正常
